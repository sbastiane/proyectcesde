FRONT-END

HTML

es un lenguaje de marcado utilizado para crear y estructurar contenido en la web. Se compone de etiquetas que definen elementos dentro de una página, como encabezados, párrafos, enlaces, imágenes y otros elementos

CSS

es un lenguaje de diseño utilizado en desarrollo web para aplicar estilos y presentación a documentos HTML. Con CSS, se pueden definir aspectos visuales como colores, fuentes, márgenes, tamaños y posiciones de elementos en una página web.

Javascript

JavaScript es un lenguaje de programación de alto nivel, interpretado y orientado a objetos. Principalmente utilizado para el desarrollo web, permite la creación de contenido dinámico e interactivo en páginas web. Como un componente fundamental de las tecnologías web, como HTML y CSS.

Angular

Angular es framework de desarrollo de código abierto y mantenido por Google. Se utiliza para construir aplicaciones web dinámicas y de una sola página (SPA). Angular facilita la creación de aplicaciones escalables y mantenibles mediante la implementación de patrones de diseño

React

React es una biblioteca de JavaScript de código abierto desarrollada por Facebook. Se utiliza para construir interfaces de usuario (UI) interactivas y reactivas en aplicaciones de una sola página (SPA). React se centra en la creación de componentes reutilizables que representan partes específicas de la interfaz de usuario.

BACK-END

Java

Java es un lenguaje de programación de propósito general que se caracteriza por ser orientado a objetos, portable y de alto rendimiento. Las características clave de Java incluyen su capacidad de ejecutarse en múltiples plataformas (portabilidad)

Spring

Spring es un extenso y modular framework de desarrollo de aplicaciones para el lenguaje Java. Creado para abordar desafíos en el desarrollo empresarial, Spring proporciona una arquitectura flexible y una amplia gama de características para simplificar el desarrollo de software en Java.

DATA BASES

MySQL

MySQL es un sistema de gestión de bases de datos relacional de código abierto, MySQL es conocido por su rendimiento y escalabilidad. Puede gestionar grandes conjuntos de datos y es utilizado en una variedad de aplicaciones, desde sitios web pequeños hasta sistemas empresariales de alta carga.

SQL Server

es una herramienta de administración y desarrollo proporcionada por Microsoft para trabajar con servidores de bases de datos Microsoft SQL Server. Facilita tareas como la creación y administración de bases de datos, la ejecución de consultas SQL yla creación y modificación de procedimientos almacenados
